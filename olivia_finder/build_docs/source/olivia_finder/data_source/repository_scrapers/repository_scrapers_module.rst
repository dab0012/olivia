Subpackages olivia_finder.data_source.repository_scrapers
=========================================================

.. automodule:: olivia_finder.data_source.repository_scrapers

Submodules description
----------------------

* **olivia_finder.data_source.repository_scrapers.cran**

   Contains the class CranScraper, which scrapes the CRAN website for package data.

* **olivia_finder.data_source.repository_scrapers.bioconductor**

   Contains the class BiocScraper, which scrapes the Bioconductor website for package data.

* **olivia_finder.data_source.repository_scrapers.r**

   Contains the class RScraper, which contains common methods for scraping the R derivate websites as (CRAN, Bioconductor)

* **olivia_finder.data_source.repository_scrapers.npm**

   Contains the class NpmScraper, which scrapes the NPM website for package data.

* **olivia_finder.data_source.repository_scrapers.pypi**
   
   Contains the class PypiScraper, which scrapes the PyPI website for package data.
