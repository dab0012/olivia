    class olivia_finder.data_source.data_source.DataSource(name: str, description: str)[source]¶

    Bases: DataSourceABC

    Base class for data sources, it implements the DataSourceABC interface. This class is not intended to be used directly, but to be inherited by other classes that must override this class’s methods.
