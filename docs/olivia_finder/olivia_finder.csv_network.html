olivia_finder.csv_ds module¶

class olivia_finder.csv_ds.CSVDataSource(name: Optional[str] = None, description: Optional[str] = None)[source]¶

Bases: DataSource

Class that implements the methods for loading a network from a CSV file. Implements the DataSource interface.

Examples

>>> from olivia_finder.csv_ds import CSVDataSource
>>> csv_ds = CSVDataSource()
>>> csv_ds.load_data("data.csv", "dependent", "dependency")
>>> package_names = csv_ds.obtain_package_names()
